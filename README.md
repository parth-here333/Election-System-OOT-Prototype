# 🗳️ Election Management System (OOT)

## 📌 Overview

The **Election Management System** is an Object-Oriented Technology (OOT) based project designed to simulate and manage the complete election process digitally. It provides multiple portals for different stakeholders like voters, candidates, administrators, and officers, ensuring transparency and efficiency.

---

## 🎯 Aim of the Project

To develop a secure and modular system that manages:

* Voter registration and verification
* Candidate nominations
* Election scheduling and monitoring
* Voting process (EVM simulation)
* Result management

---

## 🧩 Key Modules

### 👤 Voter Portal

* Register and verify voter details
* Check voting eligibility
* View assigned polling station

### 🧑‍💼 Candidate Portal

* File nominations
* Upload affidavit details
* View application status

### 🛠️ Admin Portal

* Manage voters and candidates
* Schedule elections
* Monitor system activity

### 🧑‍✈️ Officer Portal

* Manage polling stations
* Oversee election operations

### 🗳️ EVM (Electronic Voting Machine)

* Secure voting interface
* Prevent duplicate voting

### 🏢 Polling Station Portal

* Assign voters
* Track voting activity

### ⚠️ Complaint Portal

* Register and track complaints

### 🏛️ Political Party Portal

* Manage party details and candidates

### 🌍 Constituency Portal

* Manage constituency-wise data

## ▶️ How to Run

```bash

```

```bash

```

